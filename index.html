DATA COLLECTION

We do not collect any personal data. However, we utilize third-party services that collect non‑personal anonymous data to monitor and improve app performance and usability.

Firebase Services

  • Firebase Analytics: collects anonymized usage data (e.g., session duration, event interactions). IP addresses are anonymized by default :contentReference[oaicite:1]{index=1}.
  • Firebase Crashlytics: collects anonymous crash reports.
  • Firebase Remote Config: allows us to remotely adjust app behavior without an update.

All Firebase data is encrypted in transit (via HTTPS) and, for most services, also at rest :contentReference[oaicite:2]{index=2}. Access to data is strictly controlled and logged :contentReference[oaicite:3]{index=3}.

RevenueCat

We use RevenueCat to manage subscriptions and purchases. It collects:

  • Purchase history: details of initial purchases and renewals, required to ensure entitlements and receipts.
  • Anonymous app‑specific user ID: generated by RevenueCat; not linked to personal identity unless you explicitly set a user ID :contentReference[oaicite:4]{index=4}.

Collected data does not include personal identifiers, contact details, or sensitive information. All data is encrypted in transit :contentReference[oaicite:5]{index=5}.

HOW WE USE DATA

  • Improve app stability and user experience.
  • Detect and fix crashes.
  • Manage and verify subscription access.
  • Update app behavior remotely.

MOBILE APPS

Our apps are distributed via third-party app stores, which may collect device/account metadata. We do not collect location data, contact lists, or personal identifiers beyond the anonymous IDs used by Firebase and RevenueCat. Push notifications are solely opt‑in and used for app‑related updates.
