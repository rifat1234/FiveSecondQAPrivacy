DATA COLLECTION

We do not collect any personal data. However, we use third-party services that collect non‑personal anonymous data to improve usability and performance.

Firebase Services

  • Firebase Analytics: collects anonymized usage data (e.g., session length, feature interactions) to help us improve app performance and user experience.
  • Crashlytics: gathers anonymous crash reports to identify and fix app crashes.
  • Remote Config: fetches remote configuration parameters to customize app behavior without requiring an update.

All Firebase data is anonymous, does not include personal identifiers, and complies with Google’s privacy standards :contentReference[oaicite:4]{index=4}.

RevenueCat

We use RevenueCat to manage subscription and purchase data. It collects:

  • Purchase history (initial purchases, renewals)—required to verify and manage subscriptions.
  • An anonymous app‑specific user ID (not linked to your identity).

No personal or contact information is collected. Data is encrypted in transit and stored on secure servers :contentReference[oaicite:5]{index=5}.

HOW WE USE DATA

  • Enhance app functionality and user experience.
  • Monitor stability and fix crashes.
  • Manage and verify subscriptions.
  • Configure app behavior via remote settings.

MOBILE APPS

Apps are distributed through third-party stores. We do not collect location data, address book info, or identifiers beyond the anonymous IDs used by Firebase and RevenueCat. Push notifications are opt‑in and used only for app-related messaging.
